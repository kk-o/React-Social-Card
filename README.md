# React Social Card

Simple social card built with React and create-react-app. Be sure to read up and learn more about React at the official docs [here](https://reactjs.org/docs/getting-started.html).

# Installation 
`npm install`


# Usage 

### NPM Users

`npm start`

### Yarn Users

`yarn start`
